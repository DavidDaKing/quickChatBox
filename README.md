# quickChatBox
Built this quick chat box for a local host b/c I noticed that I sometimes need to send links back and forth between machines in my lab. 

# DISCLAIMER : NOT FOR PRODUCTION. USE ONLY FOR INTERNAL COMMUNICATION BETWEEN MACHINES WITHIN LAN. 

# USAGE
create a python venv 
pip3 install flask --> Or any other modules that need installing 
python3 app.py --> visit on http://127.0.0.1 this runs on port 80
